Resources

Posted on June 18th, 2018

This post contains resources and tools for stuff I'm interested in, or in general use. Last update: 09.07.2020

Disclaimer

None of these are paid promotions. I also cannot guarantee that the software is safe1 -- I also take no responsibility for anything that happens if you install any of the programs or plugins, or visit the URLs. Use common sense when installing stuff.

1: They are theoretically safe (otherwise I wouldn't use them), but as with other programs, shit does happen that wasn't intended

General tools

  • LibreOffice

Graphic design

Tools

  • Krita - open-source drawing software.
  • Inkscape - free vector art software.

Programming

Tools

  • regex101.com - Useful for regex validation.
  • Hugo - static page generator. Also powers this website
  • Vim - Currently my main and favorite editor.

Programming utilities

General/meta

  • tmux - split terminal windows. Incredibly useful with SSH or for consoles without support for tabs.

C++

  • Conan - package manager.
  • Ninja - My preferred CMake generator.
  • SCons - Open-source python-based build system.

Webdev

  • SASS - Makes CSS coding a lot easier, especially with variables and imports.

Vim

Plugins

Note that these are some select plugins I believe deserve some extra attention. The full list of plugins I use is available in my .vimrc.

  • Vim Plug - Like Vundle, but better and async.
  • NERDTree - File trees to get the IDE feel.
  • Goyo - focus enhancer
  • Limelight.vim - along with Goyo, it gives a really nice text writing experience.
  • Vimwiki - perfect for notes, as well as wiki-style personal notes/documentation.
  • Startify - beautiful start page

Themes

  • Papercolor - paper-like theme with a nice color scheme.

Firefox plugins

  • ClearURLs - big extension to remove several types of tracking, including tracking in Google results
  • DecentralEyes - loads common JavaScript plugins from a local cache instead of using certain CDNs
  • CanvasBlocker - helps block certain types of canvas-based fingerprinting techniques.
  • UA switcher and manager - changing the UA to a popular one helps make fingerprinting harder
  • Don't touch my tabs! - adds rel=noopener to all cross-site URLs
  • Redirect AMP to HTML - Prevents google from taking more control of the internet
  • BitWarden - password manager
  • uBlock Origin - blocks ads, trackers, annoying elements, and whatever else I want to go away
  • Image search options - makes it easy to search images and gifs across several sites
  • HTTPS Everywhere - Enforces HTTPS
  • Multi-account containers - helps separate sessions
