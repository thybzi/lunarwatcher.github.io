Matrix mouse picking

July 18th, 2021

Post type: Note Tags: #c++, #mouse picking, #opengl

glm::unProject is a fucking lifesaver. For it to work properly, GL_DEPTH_TEST has to be enabled. This allows glGetPixels() to return a value != 1 for GL_DEPTH_COMPONENT, which is necessary to unproject the correct coordinates. The documentation for unProject is trash. It claims the second parameter is a model-view matrix (model * view, or as I like to call it, transMatrix * camMatrix), or just a model matrix. Obligatory reminder that the model matrix is primarily used to transform the position of an object, which doesn't really matter.
